## 安装

Using npm:

```
npm install dplayer --save
```

Using Yarn:

```
yarn add dplayer
```

## Quick Start


DPlayer 有丰富的参数可以自定义你的播放器实例

| 名称                 | 默认值                             | 描述                                                                                                    |
| -------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------- |
| container            | document.querySelector('.dplayer') | 播放器容器元素                                                                                          |
| live                 | false                              | 开启直播模式, 见[#直播](#直播)                                                                          |
| autoplay             | false                              | 视频自动播放                                                                                            |
| theme                | '#b7daff'                          | 主题色                                                                                                  |
| loop                 | false                              | 视频循环播放                                                                                            |
